# Getswyft Support Documentation

Welcome to the Getswyft support documentation. Getswyft is a multi-tenant SaaS communications platform built for real estate professionals. This documentation covers every feature of the platform — from initial setup and daily messaging workflows to administration, analytics, and AI-powered automation. Use the guides below to find what you need based on your role or topic.
